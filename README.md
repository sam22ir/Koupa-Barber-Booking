**Book your favorite barber with a tap — cash on arrival in Algerian Dinars (DZD)**

📱 Android Exclusive • 🇩🇿 Algeria Market • 💵 Cash Only (DZD) • 🌐 Arabic + English

## ✨ Features
- 📞 Phone Auth via Firebase OTP
- 🔍 GPS-based barbershop search
- 📅 Easy 3-step booking
- 🔔 Real-time push notifications
- 🌍 Arabic (RTL) + English
- 🏪 Customer & Barber interfaces
- 📍 All 58 Algerian wilayas
- 💵 Cash payment only (DZD)

## 🛠️ Tech Stack
- **Supabase** — PostgreSQL 15 + Edge Functions
- **Firebase** — Phone Auth + FCM
- **Kotlin + Jetpack Compose** — Modern Android
- **MVVM + Clean Architecture**
- **Hilt** — Dependency Injection
- **PostGIS** — Geospatial queries

## 🚀 Getting Started
1. Clone the repo
2. Open in Android Studio
3. Add google-services.json to app/
4. Run: ./gradlew assembleDebug
