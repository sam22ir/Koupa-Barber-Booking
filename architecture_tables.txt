=== TABLE 13 ===
Step | Actor | Action
1 | User | Enters +213 phone number on auth_phone_entry screen
2 | Firebase Auth | Sends 6-digit OTP via SMS to Algerian number
3 | User | Enters OTP on auth_otp_verification screen
4 | Firebase Auth | Issues Firebase ID token on success
5 | App → Supabase | ID token sent to Edge Function → create/login user in Supabase
6 | App | Supabase session established → navigate to role_selection or home
